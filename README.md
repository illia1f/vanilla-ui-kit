# Vanilla UI Kit

A lightweight, framework-free UI component library built with vanilla HTML, CSS, and JavaScript.

## Project Structure

```
vanilla-ui-kit/
├── components/          # Individual UI components
│   ├── button/
│   ├── card/
│   ├── modal/
│   └── ...
├── src/                 # Source files (shared utilities, base styles)
│   ├── styles/
│   │   ├── base.css     # Reset, typography, variables
│   │   └── utilities.css
│   ├── scripts/
│   │   └── utils.js     # Shared JavaScript utilities
│   └── assets/          # Shared assets (fonts, icons, etc.)
├── dist/                # Compiled/bundled output
│   ├── ui-kit.css
│   └── ui-kit.js
├── examples/            # Component demos and examples
│   ├── index.html
│   └── examples/
├── docs/                # Documentation
│   ├── getting-started.md
│   ├── components/
│   └── guides/
└── package.json         # Tooling and scripts
```
